Sesión 15: Muestreo y Distribuciones Muestrales

🎯 Objetivos

  • Distinguir población de muestra
  • Comprender el concepto de distribución muestral
  • Aplicar el Teorema Central del Límite
  • Calcular probabilidades sobre medias muestrales

📚 Conceptos Básicos

POBLACIÓN (N): Conjunto completo de elementos que queremos estudiar.

MUESTRA (n): Subconjunto de la población seleccionado para estudio.

¿Por qué muestrear?
  • La población puede ser infinita o inaccesible
  • Es más económico y rápido
  • A veces es destructivo (pruebas de calidad)

DISTRIBUCIÓN MUESTRAL: Distribución de probabilidad de un estadístico (ej: x̄) calculado en TODAS las posibles muestras del mismo tamaño.

⭐ Teorema Central del Límite (TCL)

TEOREMA CENTRAL DEL LÍMITE:

Si tomamos muestras de tamaño n de una población con media μ y desviación σ, la distribución de las medias muestrales x̄ tiende a una normal cuando n es grande:

\[ \bar{X} \sim N\left(\mu, \frac{\sigma}{\sqrt{n}}\right) \]
Condiciones:
  • n ≥ 30 (regla práctica) O población ya normal
  • Muestreo aleatorio

Implicación: La media x̄ se distribuye normalmente AUNQUE la población NO sea normal.

Ejercicios

Ej 1: Población: alturas en Vigo con μ = 170 cm, σ = 10 cm. Tomamos muestras de n=36 personas. ¿Cómo se distribuye x̄?

Ver solución

Aplicación del TCL:

n = 36 ≥ 30 → Podemos aplicar TCL

Distribución de x̄:

\[ \bar{X} \sim N\left(170, \frac{10}{\sqrt{36}}\right) = N(170, 1.667) \]

Interpretación:

  • Media de x̄: E(x̄) = μ = 170 cm
  • Desviación típica de x̄: σ_x̄ = σ/√n = 10/6 ≈ 1.667 cm
  • La media muestral se concentra mucho más alrededor de μ que los datos individuales

Ej 2: Con los datos anteriores, calcular P(168 < x̄ < 172).

Ver solución completa

Datos: x̄ ~ N(170, 1.667)

Tipificación (estandarización):

\[ Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} = \frac{\bar{X} - 170}{1.667} \]

Límite inferior (x̄ = 168):

\[ Z_1 = \frac{168 - 170}{1.667} = \frac{-2}{1.667} \approx -1.20 \]

Límite superior (x̄ = 172):

\[ Z_2 = \frac{172 - 170}{1.667} = \frac{2}{1.667} \approx 1.20 \]

Probabilidad:

P(168 < x̄ < 172) = P(-1.20 < Z < 1.20)
= P(Z < 1.20) - P(Z < -1.20)
= 0.8849 - 0.1151 = 0.7698 ≈ 77%

Interpretación: Hay 77% de probabilidad de que la media de una muestra de 36 personas esté entre 168 y 172 cm.

Ej 3 (Contexto Vigo): Gasto mensual en transporte público (Vitrasa) de estudiantes: μ = 45€, σ = 12€. Muestra de 50 estudiantes. ¿Probabilidad de que x̄ > 48€?

Ver solución

Distribución de x̄:

\[ \bar{X} \sim N\left(45, \frac{12}{\sqrt{50}}\right) = N(45, 1.697) \]

Tipificación:

\[ Z = \frac{48 - 45}{1.697} = \frac{3}{1.697} \approx 1.77 \]

Probabilidad:

P(x̄ > 48) = P(Z > 1.77) = 1 - P(Z ≤ 1.77)
= 1 - 0.9616 = 0.0384 ≈ 3.8%

Interpretación: Solo 3.8% de probabilidad de que una muestra de 50 estudiantes tenga gasto medio superior a 48€. Sería un resultado poco común si μ realmente es 45€.

Ej 4: Si duplicamos el tamaño muestral (de n=36 a n=72), ¿cómo afecta a σ_x̄?

Ver análisis

Situación original (n=36):

σ_x̄₁ = 10/√36 = 10/6 ≈ 1.667 cm

Duplicando muestra (n=72):

σ_x̄₂ = 10/√72 = 10/8.485 ≈ 1.179 cm

Comparación:

Reducción = 1.667/1.179 ≈ 1.41 veces

Conclusión:

  • Duplicar n NO duplica la precisión
  • Reduce el error por factor √2 ≈ 1.41
  • Ley de rendimientos decrecientes: Cada vez cuesta más mejorar precisión
  • Para reducir error a la mitad, necesitamos n × 4

💡 Conceptos Clave

  • x̄ es variable aleatoria: Varía entre muestras
  • TCL es potentísimo: Funciona SIEMPRE si n es grande, sin importar la distribución poblacional
  • Mayor n → menor variabilidad de x̄: Muestras grandes son más fiables
  • Error estándar: σ/√n mide dispersión de x̄
