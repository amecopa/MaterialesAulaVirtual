Sesión 17: Intervalos de Confianza para la Media

🎯 Objetivos

  • Aplicar fórmulas de IC según si σ es conocida o no
  • Usar distribución t de Student cuando n < 30 y σ desconocida
  • Interpretar correctamente IC en contextos reales
  • Decidir tamaño muestral según precisión deseada

📚 Casos para IC de μ

Distribución t de Student:
  • Similar a la normal pero con colas más pesadas
  • Depende de grados de libertad (gl) = n - 1
  • Cuando n → ∞, t → Z (converge a normal)
  • Más conservadora que Z (ICs más amplios)

Ejercicios

Ej 1: Muestra de 15 notas de Matemáticas: x̄ = 6.8, s = 1.5. Población normal. IC al 95%.

Ver solución completa

Identificación del caso:

  • n = 15 < 30 → Muestra pequeña
  • σ desconocida (usamos s)
  • Población normal (dato clave)
  • → Usar distribución t

Grados de libertad:

gl = n - 1 = 15 - 1 = 14

Valor crítico t:

t₀.₀₂₅,₁₄ ≈ 2.145 (tabla t-Student)

Error estándar:

SE = s/√n = 1.5/√15 = 1.5/3.873 ≈ 0.387

Margen de error:

E = t · SE = 2.145 × 0.387 ≈ 0.830

IC:

IC = [6.8 - 0.830, 6.8 + 0.830] = [5.97, 7.63]

Interpretación: Con 95% de confianza, la nota media poblacional está entre 5.97 y 7.63.

Nota: Si hubiéramos usado erróneamente Z = 1.96:

E_incorrecto = 1.96 × 0.387 ≈ 0.759

IC sería [6.04, 7.56] → Más estrecho pero MENOS fiable (error metodológico).

Ej 2 (Contexto Vigo): Tiempo de viaje Vigo-Santiago en Renfe. Muestra de 50 trayectos: x̄ = 52.3 min, s = 4.8 min. IC al 99%.

Ver solución

Identificación:

  • n = 50 ≥ 30 → Muestra grande
  • σ desconocida pero n grande
  • → Podemos usar Z (aproximación normal)

Datos:

n = 50, x̄ = 52.3, s = 4.8, z₀.₀₀₅ = 2.576 (99%)

Cálculos:

SE = 4.8/√50 = 4.8/7.071 ≈ 0.679 min
E = 2.576 × 0.679 ≈ 1.75 min

IC:

IC = [52.3 - 1.75, 52.3 + 1.75] = [50.55, 54.05] minutos

Conclusión práctica: Con altísima confianza (99%), el tiempo medio de viaje está entre 50.5 y 54 minutos.

Ej 3: Peso de naranjas. Muestra de 8: datos = {180, 175, 190, 185, 172, 195, 188, 182} gramos. IC al 90%.

Ver solución completa

Paso 1: Calcular estadísticos muestrales

n = 8
Σx = 180+175+190+185+172+195+188+182 = 1467
x̄ = 1467/8 = 183.375 g

Varianza muestral:

s² = [Σ(x - x̄)²]/(n-1)
= [(180-183.375)² + (175-183.375)² + ... + (182-183.375)²]/7
= [11.39 + 70.14 + 43.89 + 2.64 + 129.39 + 135.14 + 21.39 + 1.89]/7
= 415.875/7 ≈ 59.41
s = √59.41 ≈ 7.71 g

Paso 2: Identificar caso

  • n = 8 < 30
  • σ desconocida
  • Asumimos población normal
  • → Distribución t con gl = 7

Paso 3: Valor crítico

t₀.₀₅,₇ ≈ 1.895 (90% confianza)

Paso 4: IC

SE = 7.71/√8 = 7.71/2.828 ≈ 2.73
E = 1.895 × 2.73 ≈ 5.17
IC = [183.375 - 5.17, 183.375 + 5.17]
IC = [178.2, 188.5] gramos

Ej 4 (Decisión): Dos ICs para la misma media: A=[45.2, 54.8], B=[42.1, 57.9]. Sin más información, ¿cuál tiene mayor nivel de confianza?

Ver razonamiento

Análisis de amplitudes:

Amplitud IC_A = 54.8 - 45.2 = 9.6
Amplitud IC_B = 57.9 - 42.1 = 15.8

IC_B es MÁS AMPLIO

Razonamiento:

  • Mayor confianza requiere "echar una red más amplia"
  • IC más amplio → más seguridad de capturar μ
  • Conclusión: IC_B tiene MAYOR nivel de confianza

Ejemplo numérico:

Si ambos vienen de la misma muestra:

  • IC_A podría ser 90% (z=1.645)
  • IC_B podría ser 99% (z=2.576)

💡 Guía de Decisión

¿Qué distribución uso?
  1. ¿Conozco σ? SÍ → Z
  2. ¿n ≥ 30? SÍ → Z (aunque σ desconocida, usar s)
  3. ¿n < 30 Y población normal? SÍ → t-Student
  4. ¿n < 30 Y población NO normal? Problema: necesitamos más datos o técnicas avanzadas
