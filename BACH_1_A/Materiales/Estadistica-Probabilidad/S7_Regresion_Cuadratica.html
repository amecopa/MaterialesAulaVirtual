Sesión 7: Regresión Cuadrática (Parabólica)

🎯 Objetivos

  • Reconocer situaciones donde el ajuste lineal es insuficiente
  • Comprender el modelo cuadrático y = ax² + bx + c
  • Comparar modelos lineal vs cuadrático mediante R²
  • Interpretar parámetros de la parábola (vértice, concavidad)

📚 Teoría

Regresión Cuadrática: Ajuste de una parábola a los datos

\[ y = ax^2 + bx + c \]
Parámetros:
  • a: Concavidad (a > 0 → ∪, a < 0 → ∩)
  • b, c: Ajustan posición del vértice
  • Vértice: \(x_v = -\frac{b}{2a}\)

¿Cuándo usar regresión cuadrática?

  • Datos muestran patrón curvo (no monotónico)
  • Existe un máximo o mínimo en el rango observado
  • Fenómenos cíclicos: temperatura anual, tráfico diario, etc.
  • R² del modelo lineal es bajo pero visualmente hay patrón
⚠️ Cuidado con la extrapolación: Las parábolas crecen/decrecen rápidamente fuera del rango observado. Predictiones fuera del intervalo pueden ser absurdas.

Ejercicios

Ej 1 (Contexto Vigo): Pasajeros mensuales Puerto de Vigo (cruceros + barcos Cíes) 2024-2025. Comparar R²:

  • Modelo Lineal: R² = 0.12 (muy pobre)
  • Modelo Cuadrático: R² = 0.89 (excelente)

¿Por qué la diferencia? ¿Cuál usar?

Ver análisis

Explicación del fenómeno:

  • Los pasajeros tienen patrón estacional: máximo en verano (julio-agosto), mínimo en invierno
  • Una recta no puede capturar este patrón cíclico → R² muy bajo
  • La parábola sí describe el "pico" de verano → R² alto

Decisión: Usar modelo cuadrático para predicciones dentro del año observado.

Limitación: No usar para predictos a 2-3 años vista sin ajustar, porque la parábola no captureará múltiples ciclos (para eso necesitaríamos modelos sinusoidales).

Ej 2: Parábola ajustada: \(y = -0.5x^2 + 6x + 2\)

a) ¿Es una parábola cóncava o convexa?
b) Calcular el vértice (valor x donde alcanza máximo/mínimo)
c) ¿En qué valor de x alcanza el extremo?

Ver solución completa

a) Concavidad:

a = -0.5 < 0 → Parábola cóncava (∩) → tiene un máximo

b) Vértice (coordenada x):

\[ x_v = -\frac{b}{2a} = -\frac{6}{2 \cdot (-0.5)} = -\frac{6}{-1} = 6 \]

c) Valor máximo:

\[ y_v = -0.5(6)^2 + 6(6) + 2 = -0.5(36) + 36 + 2 = -18 + 36 + 2 = 20 \]

Interpretación: La función alcanza su máximo de 20 cuando x = 6.

Ej 3 (Comparativa): Para datos de temperatura vs ventas helados:

ModeloEcuaciónR²
Linealy = 12x + 500.78
Cuadráticoy = -0.3x² + 18x + 100.94

¿Cuál elegir y por qué? ¿Qué indica el coeficiente negativo de x² en el modelo cuadrático?

Ver análisis

Elección: Modelo cuadrático (R² = 0.94 vs 0.78)

Interpretación del coeficiente -0.3:

  • La parábola es cóncava → tiene un máximo
  • Significado práctico: Las ventas aumentan con temperatura hasta cierto punto, luego empiezan a bajar
  • Posibles razones:
    • Con calor extremo (>35°C) la gente se queda en casa con aire acondicionado
    • O hay saturación de mercado

Vértice (temperatura óptima):

\[ x_v = -\frac{18}{2(-0.3)} = -\frac{18}{-0.6} = 30°C \]

Las ventas máximas ocurren alrededor de 30°C.

💡 Síntesis

  • Lineal: Relaciones monótonas (siempre creciente o decreciente)
  • Cuadrático: Cuando hay un máximo/mínimo o patrón curvo
  • Criterio numérico: Comparar R² de ambos modelos
  • Criterio gráfico: Visualizar nube de puntos + ambas curvas
